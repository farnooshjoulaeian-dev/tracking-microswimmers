# trackFar

Python software for tracking Chlamydomonas cells in microscopy videos.

## Goal

The goal is to build a reproducible pipeline for:

1. Loading microscopy videos
2. Preprocessing frames
3. Segmenting cells
4. Detecting cell centroids
5. Linking trajectories
6. Computing velocity, orientation, MSD, and MSAD
7. Visualizing results

## Current stage
